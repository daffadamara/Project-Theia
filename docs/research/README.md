# Reference Gate

Any Theia change that introduces or materially changes physics or mathematics
must have an approved research note before implementation begins. The note must
record primary sources, equations, quantities and units/normalization, boundary
conditions, invariants, parameter mapping, limitations, attribution, and license
status. Secondary implementations may validate a port but cannot replace the
original paper, official documentation, or algorithm author's publication.

An audit is approved only after the equations and stated invariants are mapped
to executable tests. If the reference or license cannot be verified, the work
remains blocked at the research gate.
